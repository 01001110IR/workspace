HTML Tags Example

In this example, you'll learn about various HTML tags and their usage:

<h1> - <h6> (Headings)

Headings are used to define headings for sections. There are six levels, with <h1> being the highest and most important. Headings provide structure and hierarchy to your content.

<p> (Paragraph)

The <p> tag is used to define a paragraph of text. Paragraphs are a fundamental way to organize and present textual content.

<a> (Anchor)

The <a> tag is used to create hyperlinks to other web pages or resources. Hyperlinks allow users to navigate between different pages or websites. For example, Click here to display the code for this page. .

<img> (Image)

The <img> tag is used to display images on a web page. Images enhance the visual appeal of your content and can convey information more effectively. For example, .

<ul> / <ol> / <li> (Lists)

The <ul> tag is used to create an unordered list, <ol> for an ordered list, and <li> for list items:

  • Unordered Item 1
  • Unordered Item 2
  • Unordered Item 3
  1. Ordered Item 1
  2. Ordered Item 2
  3. Ordered Item 3

Lists provide a structured way to present items or information in a sequence.

<table> / <tr> / <td> (Tables)

The <table> tag is used to create a table, <tr> for table rows, and <td> for table data cells:

Row 1, Cell 1 Row 1, Cell 2
Row 2, Cell 1 Row 2, Cell 2

Tables are used to organize and display tabular data in rows and columns.
